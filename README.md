# Oh-My-Pomodoro

원하는 gif, 영상을 넣어 만드는 전체화면 뽀모도로 데스크톱 앱

## 주요 기능

- 작업/휴식 전환 시 사용자가 선택한 gif 또는 영상을 **전체화면으로 재생**
- 항상 최상단에 떠 있는 미니 타이머 (드래그 이동 가능)
- 로컬 파일 또는 URL로 영상/gif 등록
- ESC로 영상 스킵, 최대 5초 자동 종료

## 기술 스택

- Electron 42 + React 19 + TypeScript
- electron-vite (Vite 기반 빌드)

## 실행

```bash
npm install
npm run dev
```

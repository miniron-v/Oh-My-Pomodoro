# Oh-My-Pomodoro

https://github.com/user-attachments/assets/80eb0a52-1130-4c39-8acc-c40abd7931cd

원하는 gif, 영상을 넣어 만드는 전체화면 뽀모도로 데스크톱 앱

## 주요 기능

<img width="591" height="506" alt="image" src="https://github.com/user-attachments/assets/ca61e106-0aad-4a83-bd76-8686265994e3" />

- **뽀모도로 타이머** — 작업 / 짧은 휴식 / 긴 휴식 자동 사이클
- **전체화면 미디어 재생** — 작업↔휴식 전환 시 사용자가 선택한 gif 또는 mp4를 전체화면으로 재생
- **미니 타이머** — 항상 최상단에 떠 있는 드래그 가능한 타이머 (리사이징 가능)
- **일시정지 / 재개 / 정지** — 타이머 제어
- **ESC 스킵** — 영상 재생 중 ESC로 즉시 스킵
- **설정** — 작업/휴식 시간, 시작/종료 영상, 알림음 모드 설정

## 다운로드

[최신 버전 다운로드](https://github.com/miniron-v/Oh-My-Pomodoro/releases/latest)

1. `Oh-My-Pomodoro-x.x.x-win.zip`을 다운로드합니다.
2. 원하는 위치에 압축을 해제합니다.
3. `Oh-My-Pomodoro.exe`를 실행합니다.

> ⚠️ Windows 보안 경고가 뜰 수 있습니다. **"추가 정보" → "실행"**을 눌러주세요.

## 시스템 요구사항

- Windows 10 / 11
- 별도 설치 불필요

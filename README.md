# Oh-My-Pomodoro

원하는 gif, 영상을 넣어 만드는 전체화면 뽀모도로 데스크톱 앱

## 주요 기능

<p><img width="49%" alt="타이머 모드" src="https://github.com/user-attachments/assets/36002a87-536b-4959-8fc9-fd6e0b56b696" /> <img width="49%" alt="알람 모드" src="https://github.com/user-attachments/assets/126ea01c-5c43-46e7-8fc9-e0c47dbdf1dc" /></p>

### 두 가지 모드

- **타이머 모드** — 작업 시간, 짧은 휴식, 긴 휴식을 설정하면 자동으로 사이클이 돌아갑니다.
- **알람 모드** — 매 시 몇 분에 작업을 시작하고, 몇 분에 휴식을 시작할지 설정합니다. 다음 전환까지 남은 시간이 표시됩니다.

### 전체화면 영상 재생

작업↔휴식이 전환될 때, 직접 등록한 gif나 영상이 전체화면으로 재생됩니다. 투명 배경 영상도 지원합니다.

- 작업 영상 (휴식 → 작업), 휴식 영상 (작업 → 휴식)을 각각 설정
- 앱에 영상을 등록해두고 골라 쓸 수 있습니다
- 영상은 최대 5초간 재생되며, ESC를 누르면 바로 넘길 수 있습니다

### 미니 타이머

화면 위에 항상 떠 있는 작은 타이머입니다. 원하는 곳에 드래그해두면 위치와 크기를 기억합니다. 필요 없을 땐 숨기고, 작업 표시줄 아이콘으로 다시 꺼낼 수 있습니다.

## 다운로드

[최신 버전 다운로드](https://github.com/miniron-v/Oh-My-Pomodoro/releases/latest)

1. zip 파일을 다운로드합니다.
2. 원하는 위치에 압축을 해제합니다.
3. `Oh-My-Pomodoro.exe`를 실행합니다.

> Windows 보안 경고가 뜰 수 있습니다. **"추가 정보" → "실행"**을 눌러주세요.

## 시스템 요구사항

- Windows 10 / 11
- 별도 설치 불필요

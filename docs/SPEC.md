# Oh-My-Pomodoro 기획서

## 1. 프로젝트 개요

### 1.1 프로젝트명
Oh-My-Pomodoro

### 1.2 한 줄 소개
원하는 gif, 영상을 넣어 만드는 전체화면 뽀모도로 데스크톱 앱

### 1.3 타겟 사용자
장시간 컴퓨터를 사용하며, 자신만의 환경을 커스텀하는 것을 즐기는 사용자

### 1.4 핵심 차별점
- 뽀모도로 작업/휴식 전환 시 사용자가 직접 선택한 영상 또는 gif를 **전체화면으로 재생**
- 항상 데스크톱 최상단에 떠 있는 미니 타이머

---

## 2. 앱 구조

앱은 크게 **2개의 섹션(창)**으로 구성된다. 타이머 표시와 전체화면 영상 재생을 하나의 창으로 통합할지, 별도 창으로 분리할지는 개발 팀장이 기술적으로 판단한다.

### 2.1 설정 창
사용자가 뽀모도로 관련 설정을 구성하는 메인 창.

**설정 항목:**
| 항목 | 설명 | 기본값 |
|------|------|--------|
| 작업 시간 | 집중 작업 시간 (분) | 25분 |
| 짧은 휴식 시간 | 작업 사이 짧은 휴식 (분) | 5분 |
| 긴 휴식 시간 | 긴 휴식 (분) | 15분 |
| 긴 휴식 인터벌 | 몇 회 작업 후 긴 휴식인지 | 4회 |
| 시작 영상 | 휴식이 끝나고 작업 시작 시 재생할 영상/gif | - |
| 종료 영상 | 작업이 끝나고 휴식 시작 시 재생할 영상/gif | - |
| 알림음 모드 | 영상 자체 소리 사용 / 별도 알림음 사용 선택 | 영상 소리 |
| 알림음 선택 | 별도 알림음 사용 시, 드롭다운으로 내장 사운드 선택 또는 사용자 업로드 | 기본 내장음 |

**영상/gif 입력 방식:**
- 로컬 파일 업로드
- URL 입력

### 2.2 뽀모도로 창 (타이머 + 영상 재생)

#### 미니 타이머
- **형태**: 전자시계 스타일 (`00:00` 포맷)
- **표시 위치**: 데스크톱 최상단 레이어에 항상 고정 (Always on Top)
- **이동**: 사용자가 드래그하여 원하는 위치로 이동 가능

#### 전체화면 영상 재생
- **작업 시간 종료 시**: '종료 영상'을 전체화면으로 재생 -> 짧은 휴식 또는 긴 휴식 시작
- **휴식 시간 종료 시**: '시작 영상'을 전체화면으로 재생 -> 작업 시작
- **영상 스킵**: 사용자가 `ESC` 키를 눌러 영상을 스킵하고 즉시 다음 단계로 진행 가능
- **사운드**: 알림음 모드 설정에 따라 영상 자체 소리 또는 별도 알림음 재생

---

## 3. 뽀모도로 사이클 흐름

```
[작업 25분] -> 종료 영상 재생 -> [짧은 휴식 5분]
[작업 25분] -> 종료 영상 재생 -> [짧은 휴식 5분]
[작업 25분] -> 종료 영상 재생 -> [짧은 휴식 5분]
[작업 25분] -> 종료 영상 재생 -> [긴 휴식 15분]
                                      ↓
                              (1 사이클 완료)
```

- 각 전환 시점마다 전체화면 영상이 재생된다.
- 휴식 -> 작업 전환 시에는 '시작 영상'이 재생된다.
- 작업 -> 휴식 전환 시에는 '종료 영상'이 재생된다.

## 4. 프리셋 시스템 (MVP 이후)

- 프리셋 = 타이머 시간 설정 + 영상/gif 조합을 하나의 세트로 저장
- 최대 **5개** 프리셋 슬롯
  - 1개: 로파이 감성 기본 내장 프리셋
  - 4개: 사용자 자유 구성
- 프리셋 저장/적용이 간편해야 함 (원클릭 적용 등)

---

## 5. 알림음 시스템

### MVP
- 영상 자체 소리 사용 / 기본 내장 알림음 사용 중 택 1 (설정에서 선택)
- 내장 알림 사운드 1개 제공

### MVP 이후
- 내장 알림 사운드 복수 제공, 드롭다운으로 선택
- 사용자가 직접 알림음 파일 업로드 가능

---

## 6. MVP 정의

### MVP 목표
> **뽀모도로 1사이클(작업 -> 짧은 휴식 -> ... -> 긴 휴식)이 성공적으로 작동한다.**

### MVP 포함 범위
| 기능 | 포함 여부 |
|------|-----------|
| 설정 창 (시간 설정, 영상 업로드/URL) | O |
| 미니 타이머 (Always on Top, 드래그 이동) | O |
| 전체화면 영상/gif 재생 (시작/종료 각각) | O |
| ESC로 영상 스킵 | O |
| 사용자 영상/gif 업로드 및 URL 입력 | O |
| 알림음 모드 선택 (영상 소리 / 내장 알림음) | O |
| 내장 알림 사운드 1개 | O |
| 뽀모도로 1사이클 정상 작동 | O |
| 프리셋 저장/관리 (5슬롯) | X (MVP 이후) |
| 알림음 커스텀 (복수 내장, 업로드) | X (MVP 이후) |
| 통계/기록 | X (MVP 이후) |

## 7. 기술 결정 사항

| 항목 | 결정 |
|------|------|
| 플랫폼 | 데스크톱 앱 (웹으로 구동 가능하다면 웹도 고려) |
| 기술 스택 | 개발 팀장에게 위임 |
| 타이머 + 영상 창 분리 여부 | 개발 팀장에게 위임 |

## 8. 용어 정리

| 용어 | 의미 |
|------|------|
| 시작 영상 | 휴식이 끝나고 작업이 시작될 때 전체화면으로 재생되는 영상 |
| 종료 영상 | 작업이 끝나고 휴식이 시작될 때 전체화면으로 재생되는 영상 |
| 프리셋 | 타이머 시간 + 영상 조합을 하나로 묶은 설정 세트 |
| 1사이클 | (작업 + 짧은 휴식) x N회 + 긴 휴식 1회로 구성된 전체 흐름 |
| Always on Top | 다른 모든 창 위에 항상 표시되는 상태 |
